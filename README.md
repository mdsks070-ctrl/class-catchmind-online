# Class Catchmind Online

Classroom drawing quiz prototype for art activities.

## Main Files

- `class_catchmind_online.html`: online room prototype with Firebase-ready sync
- `class_catchmind.html`: local classroom version
- `index.html`: smart drawing art activity prototype

## Local Preview

Open the HTML file directly, or run a simple static server in this folder and open:

```text
http://127.0.0.1:8765/class_catchmind_online.html
```

## Firebase Note

Firebase web config is already added in `class_catchmind_online.html`.
Realtime Database still needs to be created successfully in Firebase Console before multi-device sync works.
